# 04 — 玩家指南

## 大厅

进入服务器后，使用 `/fepvp` 打开主大厅 GUI：

```
┌────────────────────────────────────────────┐
│  ⚔ 锐界竞技场                               │
├────────────────────────────────────────────┤
│  [⚔ 单挑]  [👥 团队竞技]  [🎲 快速匹配]     │
│  [🏆 排行榜] [📊 我的数据] [🎒 装备库]       │
│  [👁 观战]                                  │
├────────────────────────────────────────────┤
│  在线 N 人 / 比赛中 N 场                     │
└────────────────────────────────────────────┘
```

## 单挑（1v1）

### 方式一：房间列表

1. 在大厅点击 **⚔ 单挑** → 打开单挑房间列表
2. 点击已有房间加入，或点击 **创建房间** 自建
3. 创建房间流程：选竞技场 → 选装备 → 房间创建成功
4. 房间满员（2人）后自动开赛，或房主点击 **开始比赛** 手动开赛

### 方式二：快速匹配

1. 在大厅点击 **🎲 快速匹配**
2. 自动创建一个随机竞技场的单挑房间
3. 等待其他玩家加入

## 团队竞技（2v2/3v3/4v4）

1. 在大厅点击 **👥 团队竞技** → 打开团队房间列表
2. 点击已有房间加入，或点击 **创建房间** 自建
3. 创建房间流程：选竞技场 → 选装备 → 房间创建成功
4. 进入队伍选择界面：
   - 点击红队或蓝队加入
   - 实时显示双方人数和成员
5. 房主点击 **⚔ 开始比赛**（双方各至少 1 人）
6. 或等房间满员后自动开赛

### 团队模式

| 模式 | 说明 |
|------|------|
| **淘汰制** | 队员全部阵亡即淘汰，默认模式 |
| **击杀数制** | 死亡后 2 秒复活（保留装备 + 短暂无敌），先达到目标击杀数的队伍获胜 |

模式由管理员在创建竞技场时设置，使用 `/fepvp admin arena setmode`。

## 比赛流程

```
选装备（5秒）→ 倒计时（10秒）→ 战斗 → 结算（5秒）→ 传送回大厅
```

- 倒计时期间玩家被冻结（无法移动/攻击）
- 战斗中方块可破坏/放置（需管理员开启 `auto-restore`）
- 比赛结束后自动复原方块、归还背包
- 一方死亡或超时（默认 600 秒）即比赛结束
- 死亡后自动变为旁观者（淘汰制）或 2 秒后复活（击杀数制）

## 背包保护

- 比赛前自动备份背包（精确到 NBT 级别）
- 比赛结束自动归还原物品
- 断线重连后自动检测并恢复背包（崩溃恢复）
- 击杀数制复活时：自带装备保留，装备组合重新发放

## 方块复原

管理员开启 `protection.auto-restore` 后：

- 比赛中允许破坏和放置方块
- 比赛结束后自动复原所有被修改的方块
- 服务端崩溃后重启也会自动恢复未复原的方块

## 排行榜

在大厅点击 **🏆 排行榜**，或执行：

```
/fepvp stats          # 查看自己
/fepvp stats 玩家名   # 查看他人
```

排行榜显示：

```
═══════════ 段位排行榜 (1/3) ═══════════
🥇 #1 PlayerA  ELO: 2100 | 胜: 80 | 王者
🥈 #2 PlayerB  ELO: 1950 | 胜: 65 | 宗师
🥉 #3 PlayerC  ELO: 1800 | 胜: 50 | 大师
→ #7 You       ELO: 1500 | 胜: 30 ← 你
════════════════════════════════════════
```

## 段位体系

| 段位 | ELO 范围 |
|------|----------|
| 王者 | 2000+ |
| 宗师 | 1800-1999 |
| 大师 | 1600-1799 |
| 精英 | 1400-1599 |
| 老手 | 1200-1399 |
| 熟手 | 1000-1199 |
| 新手 | < 1000 |

## 观战

大厅点击 **👁 观战**，或执行：

```
/fepvp spectate
```

- 以旁观者模式传送至观战点
- 再次执行 `/fepvp spectate` 退出观战
- 管理员可通过 `config.yml` 关闭观战功能

## 离开房间 / 退出比赛

```
/fepvp room leave     # 离开当前房间
/fepvp leave          # 离开当前比赛
```

- 离开房间：房主离开则自动转移给下一位玩家
- 房间所有人离开后自动解散
- 比赛中退出视为投降（对手获胜）
