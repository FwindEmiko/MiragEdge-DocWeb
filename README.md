# 锐界幻境文档站 ✰ MiragEdge Docs

## <MapIcon name="book-open" :size="24" /> 项目介绍

锐界幻境（MiragEdge）是基于高版本Minecraft（Java 26.1.2）的跨平台互通服务器，支持Java版与基岩版玩家同服游玩。本仓库是服务器官方文档库，涵盖**玩家指南**、**开发协作**、**版本日志**等核心内容，旨在帮助玩家快速上手、开发者高效协作，同时记录服务器的成长历程。

锐界幻境致力于打造一个开放创新的服务器平台，为玩家提供纯净的游戏体验。我们注重技术创新、社区建设和内容创造，努力在数字世界中为每位玩家打造属于自己的幻境。

## <MapIcon name="building-2" :size="24" /> 技术栈
本项目采用现代前端技术栈构建：
```bash
Framework: VitePress 1.6.4
Language: TypeScript 5.9.3
Build: pnpm + Node.js with optimized dependencies
Visualization: Mermaid, Canvas-confetti
```

### 主要依赖
- **VitePress**: 现代静态站点生成器
- **Mermaid**: 图表和流程图渲染
- **Canvas-confetti**: 粒子动画效果
- **Axios**: HTTP客户端
- **@octokit/rest**: GitHub API客户端
