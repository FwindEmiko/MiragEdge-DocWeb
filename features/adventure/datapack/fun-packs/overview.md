# 趣味玩法 · 总览

<SmartImage src="/images/datapack/incendium/biomes/Inverted_forest_tera.png" alt="趣味玩法" shadow radius="auto" caption="从天而降的羽毛到自动护矿——点缀冒险的小彩蛋" />

> 三个小型趣味机制数据包——它们不改变世界面貌，但为游戏增添了一些意想不到的小乐趣（和小挑战）。


## Chicken Shedding · 落羽

**鸡会定期掉落羽毛** — 一个简单但实用的小机制。

| 项目 | 说明 |
|------|------|
| **效果** | 鸡会像现实一样定期掉落羽毛 |
| **频率** | 每隔一段时间自动脱落一根 |
| **影响** | 再也不用杀鸡取羽了！羽毛农场变得可行 |
| **兼容** | 不影响原版鸡的繁殖和掉落 |

> 箭矢自由的第一步——养一群鸡，躺着收羽毛！


## Nether Creepers · 炎爆

**下界闪电苦力怕** — 下界版的苦力怕，更危险也更刺激。

| 项目 | 说明 |
|------|------|
| **生成地点** | 下界各处 |
| **特殊能力** | 爆炸威力比主世界苦力怕**更大** |
| **外观** | 烈焰色的特殊纹理 |
| **危险程度** | |
| **应对建议** | 保持距离，用远程武器解决 |

> 在下界挖矿时听到"嘶嘶"声——快跑！这可不是主世界那种小爆炸。


## Possessive Pigmen · 护矿

**猪灵守护金矿** — 挖掘下界金矿会激怒附近的猪灵。

| 项目 | 说明 |
|------|------|
| **触发条件** | 在下界挖掘**金矿石**或**下界金矿石** |
| **效果** | 附近的僵尸猪灵会被激怒并攻击你 |
| **范围** | 一定半径内的所有僵尸猪灵 |
| **应对建议** | 挖金矿前先清理周围猪灵，或者穿着金甲降低敌意 |

> 金子在猪灵眼中是神圣的——动他们的金子，就得面对他们的怒火！


## 影响评估

| 数据包 | 对游戏的影响 | 影响等级 |
|--------|-------------|----------|
| 落羽 | 纯正面——免费羽毛 | 低 |
| 炎爆 | 纯挑战——下界更危险 | 中 |
| 护矿 | 增加难度——挖金矿需要策略 | 中 |
