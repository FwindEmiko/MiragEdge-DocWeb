# 地城酒馆 · 附魔列表

> D&T 新增了 **17 种**自定义附魔，全部以附魔书形式散落在特定结构的宝箱中。每种附魔都有独特的获取地点和玩法——有些能改变你的战斗方式，有些让你探索世界更加轻松。

---

## 武器附魔

### Aerial's Bane · 空灾

| 属性 | 说明 |
|------|------|
| **效果** | 对空中目标造成额外伤害——击飞、跳跃、下落的敌人通通受到加成。也对旋风人、恼鬼、凋零有效 |
| **适用** | 剑、斧、三叉戟、锤、弓、弩 |
| **最高等级** | V (5) |
| **冲突** | 锋利、力量等伤害加成附魔 |
| **获取** | 孤寂要塞 · 封印大厅 · 末地城堡 |

> 配合击退附魔使用——把敌人打上天再一刀带走！

---

### Wither Coated · 凋零涂装

| 属性 | 说明 |
|------|------|
| **效果** | 直接命中时施加**凋零效果**。等级越高，凋零伤害生效越快（总伤害相同，但持续时间缩短）每次攻击额外消耗 1 点耐久 |
| **适用** | 剑、斧、三叉戟、锤 |
| **最高等级** | III (3) |
| **冲突** | 火焰附加 |
| **获取** | 下界港口 · 封印大厅 |

> 对高血量 Boss 特别好用——持续百分比扣血无视护甲！

---

### Illager's Bane · 灾厄克星

| 属性 | 说明 |
|------|------|
| **效果** | 对灾厄村民、劫掠兽、恼鬼、女巫造成额外伤害 |
| **适用** | 剑、斧、三叉戟、锤 |
| **最高等级** | V (5) |
| **冲突** | 锋利等伤害附魔 |
| **获取** | 灾厄庄园 · 灾厄藏身处 |

> 攻打灾厄庄园和藏身处之前必带！

---

### Spiteful · 怨恨

| 属性 | 说明 |
|------|------|
| **效果** | 血量低于 2.5 颗心时获得攻击力加成。被攻击时（任意血量）有概率获得力量效果 |
| **适用** | 剑、斧、三叉戟、锤 |
| **最高等级** | III (3) |
| **冲突** | 无 |
| **获取** | 苍白居所 · 特定神殿 |

> 残血反杀神器——血量越低伤害越高！

---

## 盔甲附魔

### Antidote · 解毒

| 属性 | 说明 |
|------|------|
| **效果** | **完全免疫**中毒和凋零伤害——所有伤害转移给胸甲（消耗耐久） |
| **适用** | 胸甲 |
| **最高等级** | I (1) |
| **冲突** | 保护、火焰保护、爆炸保护、弹射物保护 |
| **获取** | 毒液巢穴 |

> 毒液巢穴探索必备！女巫和毒液史莱姆瞬间变废柴。

---

### Outreach · 延伸

| **冲突** | 无 |

## 靴子附魔

### Traveler · 旅者

| 属性 | 说明 |
|------|------|
| **效果** | 冲刺时提升移动速度和步高——可以像马一样走上 1 格高的坡。冲刺时缓慢消耗耐久 |
| **适用** | 靴子 |
| **最高等级** | III (3) |
| **冲突** | 灵魂疾行 |
| **获取** | 神殿 · D&T 结构中随机墓地 · 亡灵地穴 · 神殿塔 · 失落碉堡 |

> 赶路神附魔！不用跳就能上坡，速度还快。

## 鞘翅附魔

## 通用附魔

### Photosynthesis · 光合作用

| 属性 | 说明 |
|------|------|
| **效果** | 在**阳光直射**或能种植作物的光照下，缓慢恢复物品耐久。盔甲架穿戴时也生效 |
| **适用** | 任意有耐久的物品 |
| **最高等级** | I (1) |
| **冲突** | 经验修补 |
| **获取** | 丛林遗迹 |

> 白天下矿、砍树——工具自己慢慢回耐久，省下无数经验！

| 沙漠遗迹 | Outreach（延伸） |

| 下界要塞 | Hydro Veil、Swift Soar、Wither Coated |
| 下界港口 | Swift Soar |
