# 沧墟破门 · 结构一览

> **Hoppo** 系列包含两个数据包，分别翻新了原版的废弃传送门和海底废墟——让这些原本单调的结构变成值得探索的小型遗迹。

---

## Hoppo Better Ruined Portals · 破门

原版废弃传送门只是几块黑曜石加一个金块。**破门**将它们变成了**每个都不一样的独特遗迹**。

### 特色改进

| 改进项 | 说明 |
|--------|------|
| **多样化结构** | 每个废弃传送门都有独特的外观——不再千篇一律 |
| **环境融合** | 传送门会根据所处群系呈现不同的风格 |
| **更好的战利品** | 宝箱中的物资更加丰厚合理 |
| **叙事元素** | 每个传送门看起来都像是一个"故事"的遗迹 |

### 传送门变体（部分）

| 变体 | 描述 |
|------|------|
| **森林传送门** | 被藤蔓缠绕的石质传送门框架 |
| **沙漠传送门** | 被沙土半掩的传送门 |
| **山地传送门** | 嵌入山体的传送门遗迹 |
| **下界传送门** | 下界侧的传送门——比主世界侧更完整 |
| **水下传送门** | 沉入海底的传送门（形成小型水下遗迹） |

---

## Hoppo Better Underwater Ruins · 沧墟

原版海底废墟通常是几块石砖 + 一个箱子。**沧墟**把海底变成了真正的**水下考古现场**。

### 特色改进

| 改进项 | 说明 |
|--------|------|
| **更大更复杂** | 海底废墟的规模大幅提升 |
| **多样化设计** | 不同风格的水下建筑——从民居到神殿 |
| **丰富宝藏** | 藏宝图、海洋之心、鹦鹉螺壳等海洋主题战利品 |
| **探索体验** | 水下探索不再是"看一眼箱子就走" |

### 废墟类型

| 类型 | 描述 |
|------|------|
| **小型民居** | 沉入海底的小屋 |
| **中型建筑** | 两层楼高的石质建筑 |
| **海底神殿残骸** | 破损的海底神殿风格建筑 |
| **沉船残骸** | 更加精细的沉船变体 |
| **珊瑚覆盖遗迹** | 被珊瑚礁覆盖的古老遗迹 |
