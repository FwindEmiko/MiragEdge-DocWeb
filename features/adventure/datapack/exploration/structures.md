# 野外探索 · 小型结构一览

<SmartImage src="/images/datapack/incendium/biomes/Ash_barrens_tera.png" alt="野外探索" shadow radius="auto" caption="散落在世界各处的微型探索点——每一处都藏着惊喜" />


## ATi · 遗迹结构

**古代遗迹** ——在世界各地散布神秘的古遗迹。

| 结构名称 | 描述 |
|----------|------|
| **古代墓穴** | 深埋地下的古老墓室，可能有宝藏也可能有陷阱 |
| **石碑群** | 散布在平原上的石柱群 |
| **祭祀台** | 远古文明的祭祀遗址 |
| **小型神殿** | 被遗弃的小型神殿结构 |
| **沙漠石棺** | 沙漠中的古棺，小心里面的木乃伊 |


## Explorify · 探境

小型探索点——给每一次野外旅行加点"哦？那是什么？"的惊喜。

| 结构名称 | 描述 |
|----------|------|
| **探险者营地** | 前人留下的临时营地 |
| **路标** | 荒野中的指路石碑 |
| **小屋废墟** | 废弃的小屋，可能残存少量物资 |
| **小祭坛** | 森林深处的神秘祭坛 |
| **树屋残骸** | 倒塌的树屋 |


## Stone Rooms · 石室

**石质房间**——隐藏在山脉和地下的石室结构。

| 结构名称 | 描述 |
|----------|------|
| **山体密室** | 嵌入山体的石质房间 |
| **地下石厅** | 地下的矩形石质大厅 |
| **矿工休息室** | 矿道旁的简易休息间 |
| **石室藏宝间** | 有宝箱的隐藏石室 |


## Forgotten Hallways · 遗廊

**地下走廊**——深埋在地下的古老长廊和密室网络。

| 结构名称 | 描述 |
|----------|------|
| **地下长廊** | 长条形的走廊结构，连接多个房间 |
| **遗忘密室** | 走廊尽头的隐藏房间 |
| **图书馆残骸** | 古老的地下图书馆 |
| **地下仓库** | 堆满杂物的地下储藏室 |


## 探索建议

| 探索风格 | 推荐路线 |
|----------|----------|
| 登山探险 | 山地找石室（Stone Rooms） |
| 地下探秘 | 洞穴中找遗廊（Forgotten Hallways） |
| 平原漫步 | 平原找遗构石碑群（ATi） + 探境路标（Explorify） |
| 沙漠考古 | 沙漠找石棺和古代遗迹 |
