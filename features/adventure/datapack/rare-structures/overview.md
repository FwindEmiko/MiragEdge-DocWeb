# 稀有建筑群 · 总览

> 三个大型村庄/城市改造数据包的集合——它们将原版单调的村庄变成了风格各异、令人叹为观止的聚落。

---

## Lios Overhauled Villages · 灵村

**全面翻新原版村庄**——每一个村庄都变成了视觉盛宴。

### 特色

| 特点 | 说明 |
|------|------|
| **全群系适配** | 每种村庄群系（平原、沙漠、针叶林、雪地、热带草原）都有专属风格 |
| **建筑升级** | 房屋更大、更精致、更符合群系主题 |
| **道路改善** | 村庄道路更加自然，连接更流畅 |
| **装饰细节** | 村庄内有更多装饰——花坛、路灯、水井、市集 |
| **功能完整** | 所有原版村民工作站都包含在其中 |

---

## Lukis Grand Capitals · 大都

**宏伟的大型城市结构**——不只是村庄，而是真正的城市。

### 特色

| 特点 | 说明 |
|------|------|
| **城市规模** | 比原版村庄大数倍，有城墙、城门、街道 |
| **建筑多样性** | 包含市政厅、市场、住宅区、工匠区等多种建筑类型 |
| **高度差设计** | 城市利用地形高度差，房屋层次分明 |
| **防御工事** | 城墙、瞭望塔、护城河 |
| **独特风格** | 每种群系都有独立设计的城市风格 |

### 城市类型

| 类型 | 生成群系 | 描述 |
|------|----------|------|
| **平原大都** | 平原 | 经典中世纪风格城市 |
| **沙漠大都** | 沙漠 | 砂岩建筑的沙漠都市 |
| **针叶林大都** | 针叶林 | 木石混搭的北方城市 |
| **雪地大都** | 积雪群系 | 冰雪覆盖的白色都市 |

---

## Towns & Towers · 村镇塔楼

**新增城镇与塔楼**——在主世界中添加中型定居点和各种风格的塔楼。

### 城镇特色

| 特点 | 说明 |
|------|------|
| **中型规模** | 介于村庄和城市之间 |
| **塔楼点缀** | 城镇周围有各种塔楼——钟楼、瞭望塔、灯塔 |
| **自然融合** | 城镇布局考虑地形，不破坏自然景观 |
| **新建筑类型** | 包含原版村庄没有的建筑——铁匠铺升级、大教堂、市政厅 |
