## Terralith · 地灵

**主世界地形大修** — 新增 **100+ 个群系**，被誉为 Minecraft 最好的地形数据包。

### 核心特色

| 特点 | 说明 |
|------|------|
| **群系数量** | 100+ 全新群系覆盖主世界 |
| **地形壮丽** | 巍峨的山脉、深邃的峡谷、连绵的丘陵 |
| **原版兼容** | 完全兼容原版群系，不删除任何内容 |
| **细节丰富** | 每个群系有独特的地表方块、植被和地貌 |

### 代表群系

| 群系 | 描述 |
|------|------|
| **雪山高原** | 高耸入云的雪山，云层在山腰飘过 |
| **樱花谷** | 樱花树遍布的山谷，全年粉色花雨 |
| **火山口** | 死火山形成的环形山，内部有独特矿物 |
| **迷雾森林** | 常年笼罩在雾气中的原始丛林 |
| **沙漠峡谷** | 被河流切割的沙漠台地 |
| **热带群岛** | 散布在温暖海洋中的小岛群 |


## Tectonic · 地脉

**地质构造重制** — 更戏剧化的地形、更深的海洋、更真实的地质特征。

### 核心特色

| 特点 | 说明 |
|------|------|
| **山脉增强** | 山脉更高更陡，有真实的山脊线 |
| **海洋加深** | 海洋深度增加，海底地形更复杂 |
| **河流系统** | 更自然的河流走向和宽度 |
| **高原台地** | 平坦的高原区域，适合建造 |
| **地下结构** | 更丰富的洞穴和峡谷网络 |

> 地脉与地灵完美配合，创造出令人窒息的自然景观。


## Nullscape · 虚景

**末地地形增强** — 让末地的天空岛更加壮丽多变。

### 核心特色

| 特点 | 说明 |
|------|------|
| **天空岛变大** | 末地外岛面积大幅增加 |
| **地形起伏** | 末地不再平坦，有了丘陵和山谷 |
| **高度变化** | 不同 Y 层有不同的地貌特征 |
| **虚空美学** | 增强末地天空和虚空的视觉效果 |


## Subsurface · 洞渊

**洞穴系统扩展** — 更深、更广阔的地下世界。

### 核心特色

| 特点 | 说明 |
|------|------|
| **洞穴深度** | 洞穴延伸至更深的地层 |
| **大型洞穴** | 巨大的地下空洞和穹顶 |
| **连接系统** | 洞穴之间通过通道连接的更加自然 |
| **地下群系** | 不同深度有不同的地下生态系统 |


## U Desert · 漠谷

**沙漠结构扩展** — 为荒漠地区增添神秘建筑。

### 核心特色

| 特点 | 说明 |
|------|------|
| **沙漠遗迹** | 新增多种沙漠专属结构 |
| **绿洲** | 沙漠中罕见的绿色庇护所 |
| **沙丘地貌** | 更自然的沙漠地形——真正的沙丘 |


## 协同效果

这五个数据包在锐界幻境服务器上同时运行，产生以下效果：

| 维度 | 效果 |
|------|------|
| **主世界** | 地灵 (Terralith) + 地脉 (Tectonic) → 史诗级地形 |
| **洞穴** | 洞渊 (Subsurface) → 深邃的地下世界 |
| **沙漠** | 漠谷 (U Desert) → 丰富的沙漠探险 |
| **末地** | 虚景 (Nullscape) → 壮丽的虚空群岛 |
