## 攻击模式全解

### 1. 俯冲冲击波（Dragon Dive）
末影龙向地面俯冲，落地时产生**巨大的冲击波**——站得太近会被击飞 + 高伤害。

**应对**：看到龙俯冲时立即跑开，落地瞬间跳跃躲避冲击波。


### 2. 三连俯冲（Triple Dive）
当龙血量降至 **75% 以下**时，栖息状态下会连续俯冲 **3 次**！

**应对**：千万别贪刀！每次俯冲后立刻拉开距离，等待三次俯冲结束再输出。


### 3. 连锁龙息火球（Chain Dragon Fireball）
龙在空中连发多枚龙息火球，落地产生持续 **60 秒**的滞留伤害云（原版仅 30 秒）。

**应对**：注意脚下的紫色粒子，不要停在龙息云里。龙息火球现在**直接命中也有伤害**。


### 4. 巨型激光（Huge Laser）
龙从口中发射巨大的激光束，横扫整个战斗平台。

**应对**：看到龙嘴发光立刻躲到黑曜石柱后面，或利用末影珍珠快速位移。


### 5. 终局阶段（Final Phase）
当龙血量降至 **10%** 时：
- 龙**免疫所有伤害**直到栖息
- 如果 30 秒内未栖息，龙会强制栖息
- 栖息后龙悬浮在低空——你需要**跳起来打最后一击！**

**应对**：这是整个战斗最帅的时刻。准备好缓降药水或末影珍珠，跳上空中完成最后一刀。

## 战斗流程

### 第一阶段：摧毁末影水晶（100% - 10% HP）

1. 进入末地，先**不要**管龙——直奔末影水晶塔
2. 每座水晶塔被一只**幻翼**守卫——优先击杀幻翼
3. 击杀幻翼后铁笼破碎，摧毁末影水晶
4. 注意：龙在**所有水晶被摧毁前免疫伤害**
5. 屏幕会显示水晶剩余数量的计数器


### 第二阶段：常规战斗（100% - 10% HP）

所有水晶摧毁后，龙开始受到伤害：

- 用弓箭远程消耗
- 注意躲避俯冲冲击波和激光
- 龙栖息在传送门时是近战输出的最佳时机
- 小心龙息火球——现在持续 60 秒且直接命中也有伤害


### 第三阶段：终局对决（10% - 0% HP）

这是 True Ending 的招牌时刻：

1. 龙血量降到 10% 后**免疫伤害**
2. 龙会强制栖息——悬浮在低空，等待最后一击
3. **跳起来**用近战武器完成最后一击
4. 龙被击杀后，128 格内所有玩家获得「解放末地」成就
5. 享受胜利的 Boss 音乐！


## 设置与配置

True Ending 提供可配置的设置菜单：

```
/function true_ending:_settings
```

可调整项包括：
- 末影龙生命值
- 攻击伤害倍率（与难度挂钩——简单模式伤害降低，困难模式伤害提升）


## 其他改动

| 改动项 | 说明 |
|--------|------|
| 龙息云持续时间 | 30 秒 → **60 秒** |
| 龙息火球直接伤害 | 原版无 → **有直接命中伤害** |
| 末影珍珠 | 会被龙的新攻击**摧毁**（防止刷珍珠） |
| 末影水晶火焰 | 变为**灵魂火焰** |
| 龙复活 | 复活时播放全局提示音 |
| 龙复活动画 | 全新重制 |


## 小贴士

| 问题 | 解决方案 |

| 被冲击波击飞 | 末影珍珠回平台 / 缓降药水 |
