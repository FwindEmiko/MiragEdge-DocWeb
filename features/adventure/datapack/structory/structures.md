# 筑物塔楼 · 全建筑一览

<SmartImage src="/images/datapack/incendium/structures/Abandoned_tower.png" alt="筑物塔楼" shadow radius="auto" caption="荒野中的废墟与塔楼——每一座都有独特的故事" />

> **Structory** 系列数据包为 Minecraft 世界增添了丰富的野外小型建筑。分为 **Structory**（筑物）和 **Structory Towers**（塔楼）两个子包，总计包含数十种建筑变体，让探索世界的过程不再单调。


## Structory · 筑物

野外小型建筑的集合——废墟、营地、遗迹，让每一片森林和草原都有故事。

### 废墟与遗迹

| 建筑名称 | 英文名 | 描述 |
|----------|--------|------|
| **古旧废墟** | Old Ruins | 被时间侵蚀的石质建筑残骸 |
| **废弃哨站** | Abandoned Outpost | 曾经的前哨据点，如今只剩下断壁残垣 |
| **石质拱门** | Stone Arch | 自然侵蚀形成的天然石拱 |
| **倒塌塔楼** | Collapsed Tower | 塔楼的半截残骸，内部可能有隐藏宝箱 |
| **古代石碑** | Ancient Monolith | 来历不明的巨大石碑 |
| **流浪者营地** | Wanderer Camp | 旅人留下的临时营地残迹 |


### 自然构造

| 建筑名称 | 描述 |
|----------|------|
| **巨石阵** | 类似现实世界巨石阵的环形石柱 |
| **枯木图腾** | 凋零树木形成的天然祭坛 |
| **苔藓岩堆** | 长满苔藓的岩石堆 |


## Structory Towers · 塔楼

各种风格和用途的塔楼结构，从平原上的瞭望塔到山顶的法师塔。

### 塔楼系列

| 建筑名称 | 描述 | 常见群系 |
|----------|------|----------|
| **瞭望塔** | 简单的木质瞭望塔 | 平原、森林 |
| **石质哨塔** | 石砖搭建的坚固哨塔 | 山地、丘陵 |
| **法师塔** | 带有魔法气息的高塔 | 黑森林、沼泽 |
| **废墟塔** | 半毁的古塔 | 各群系 |
| **灯塔** | 海岸边的指引灯塔 | 海岸、沙滩 |
| **钟楼** | 带有钟的塔楼 | 平原村庄附近 |


## 探索提示

| 想找... | 去哪里 |
|----------|--------|
| 废墟 | 森林、平原、针叶林 |
| 塔楼 | 山地、丘陵、高地 |
| 营地 | 河边、湖边 |
| 巨石阵 | 平原开阔地带 |

这些结构都相对小型的点缀性建筑，不需要特别准备——但它们可能藏着意想不到的宝藏或线索。
