# 死亡轮回

当你在冒险中不幸倒下，灵叶散落一地……别慌，死亡轮回系统帮你把遗失的东西找回来。这个系统在惩罚死亡的同时，也为你提供了挽回损失的机会。

## 死亡时会发生什么

每次死亡，系统会自动记录你的死亡位置，并根据你**当天的累计死亡次数**扣除一定数量的灵叶，以发光的金粒形式掉落在死亡点。

### 阶梯惩罚

死亡惩罚不是固定的，而是根据你当天死了多少次来决定：

| 当天死亡次数 | 惩罚阶梯 | 灵叶损失范围 |
| :---: | :---: | :---: |
| 1 ~ 3 次 | 免罚 | 0 |
| 4 ~ 6 次 | 轻度 | 100 ~ 250 |
| 7 ~ 10 次 | 中度 | 250 ~ 500 |
| 11 ~ 15 次 | 重度 | 500 ~ 800 |
| 16 ~ 20 次 | 严重 | 800 ~ 1200 |
| 21 次及以上 | 最高 | 1200 ~ 1500 |

::: tip 每日重置
每天的死亡次数会在**每日零点重置**，所以新的一天又是新的开始。前 3 次死亡完全免罚，日常小失误不用担心。
:::

### 灵叶掉落

- 死亡扣除的灵叶会变成一颗**发光的金粒**掉落在死亡点，上方显示名称「野生灵叶」
- 金粒会在地上存在 **6 分钟**，超时后消失
- 金粒**不会和其他掉落物合并**，保证你能完整拾取

## 返回死亡点：/back

死亡后，你可以使用 `/back` 命令（或 `/feback`）传送回死亡点。

### 使用方式

```
/back
```

Java 版玩家在死亡后还会收到一条可点击的聊天消息，点击即可直接返回。

### 返回费用

每次返回需要支付 **400 灵叶**（从 Vault 余额扣除）。

- 拥有 `feback.free` 权限的玩家可以免费返回
- 如果灵叶不足，将无法传送

### 返回后的负面效果

传送不是毫无代价的，返回死亡点后你会短暂受到影响：

| 效果 | 持续时间 | 说明 |
|------|----------|------|
| 🍖 饥饿值减少 | -2 点 | 需要补充食物 |
| 🌫️ 失明 | 约 3 秒 | 视野受限 |
| 🐌 缓慢 | 约 4 秒 | 移动减速 |

::: warning 负面效果
这些效果很快就会消退，不影响正常游戏。但请做好准备再返回危险区域！
:::

### 冷却时间

两次 `/back` 之间需要间隔 **5 秒**。

## 灵叶拾取规则

### 拾取他人的灵叶

如果你路过别人掉落的灵叶金粒，可以直接拾取，灵叶会存入你的 Vault 余额。金粒主人会收到一条通知，告诉你谁捡走了他的灵叶。

### 拾取自己的灵叶

| 方式 | 能否拾取 | 说明 |
|------|----------|------|
| 🚶 自己走回死亡点 | <MapIcon name="check" :size="24" /> 可以 | 正常拾取 |
| ⚡ 使用 /back 传送 | ❌ 不能 | 系统提示无法拾取 |

::: info 防套利设计
这个设计是为了防止「传送回去直接捡钱」的套利行为。如果你想要回灵叶，就得自己走过去。
:::

## 基岩版玩家

如果你是通过 Geyser/Floodgate 连接的基岩版玩家：

- 死亡重生后 **2 秒**，会自动弹出一个表单，询问你是否要返回死亡点
- 表单会在 **30 秒** 后自动关闭
- 如果表单弹出失败，你可以直接输入 `/back` 命令手动返回

## 世界排除

::: danger PVP 世界禁用
在 **PVP** 世界中，死亡轮回系统完全禁用——不会记录死亡点、不会扣除灵叶、无法使用 `/back`。
:::

## 实用技巧

### 新手建议

1. **前 3 次死亡免费** — 不用担心日常失误
2. **死亡后先记位置** — 看看死亡点在哪里，决定是否值得回去
3. **保持灵叶余额** — 至少留 400 灵叶作为返回费用
4. **组队探索** — 有队友帮忙捡回来更安全

### 高手玩法

1. **利用死亡机制** — 快速传送回死亡点（虽然不能捡灵叶）
2. **灵叶投资** — 把多余灵叶存起来，以备不时之需
3. **观察他人掉落** — 有机会捡到其他玩家的灵叶

祝你在冒险中谨慎前行，少死几次！🍀
