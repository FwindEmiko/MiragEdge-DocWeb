# 黄玉护甲

冬天户外的必备护甲，由黄玉和钢锭打造。

## 制作材料

| 材料 | 数量 | 获取 |
|------|------|------|
| 黄玉 | 8个 | 高山生物群系 Y<40 挖掘 |
| 钢锭 | 24个 | 高炉烧制（铁块+煤炭） |

## 合成配方

| 部位 | 材料 |
|------|------|
| 头盔 | 黄玉×2 + 钢锭×4 |
| 胸甲 | 黄玉×2 + 钢锭×8 |
| 护腿 | 黄玉×2 + 钢锭×6 |
| 靴子 | 钢锭×4 |

## 保暖效果

| 护甲 | 全套保暖值 |
|------|-----------|
| 皮革 | +20℃ |
| 铁/金/钻石 | +5℃ |
| 下界合金 | +3℃ |
| **黄玉护甲** | **+45℃** |
