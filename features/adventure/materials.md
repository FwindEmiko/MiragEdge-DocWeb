# 新增材料

锐界幻境新增的特殊材料。

## 黄玉

稀有矿物，散发橙黄色光芒。

| 属性 | 说明 |
|------|------|
| 获取地点 | 高山生物群系 |
| 挖掘深度 | Y < 40 |
| 稀有度 | 比钻石更稀有 |
| 工具要求 | 钻石镐 |

## 钢锭

由铁块在高炉中煅烧而成。

| 属性 | 说明 |
|------|------|
| 熔炉类型 | 高炉 |
| 配方 | 铁块 + 煤炭 |
| 速度 | 比普通熔炉快2倍 |
