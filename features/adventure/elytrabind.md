# 鞘翅绑定

在锐界幻境，每个玩家终身只能拥有一个鞘翅——它从你拾起的那一刻起，就只属于你。

::: tip 核心理念
一人一鞘翅，防止囤积垄断。你的鞘翅不会掉落、不会丢失、别人捡不走。
:::

## 如何获得鞘翅

1. 前往**末地**，找到**末地城**
2. 在末地船的船头，你会看到一个物品展示框，鞘翅就挂在上面
3. **左键破坏展示框**，鞘翅会掉落在地
4. 走过去**拾取**——系统会自动把它绑定给你

拾取成功的瞬间，你会收到一条祝贺消息，鞘翅上也会出现你的名字和绑定日期。

::: warning 注意
只有**还没绑定过鞘翅**的玩家才能破坏展示框。如果你已经绑定过，展示框会被保护——请把它留给还没获得鞘翅的朋友。
:::

### 展示框保护机制

末地船上的鞘翅展示框受到特殊保护——确保每一只鞘翅都能遇到它的主人：

- **未绑定玩家**：正常破坏展示框，获取鞘翅
- **已绑定玩家**：无法破坏展示框（包括直接攻击和打掉它附着的方块）
- **爆炸、活塞、水流等外力**：任何方式都无法摧毁展示框

> 简单说：鞘翅展示框只有等待它的人才能打开，不会因为意外或恶意破坏而永远消失。

## 末地外岛飞行限制

为了保护鞘翅绑定机制不被绕过，**末地外岛禁止使用鞘翅飞行**。

- 在末地外岛区域内，鞘翅的飞行功能会被禁用
- 你可以正常装备鞘翅，但无法用烟花火箭推进起飞
- 离开末地外岛后，飞行功能自动恢复

::: tip 为什么
如果允许在末地外岛自由飞行，已经绑定的玩家可以轻松飞到其他末地城，抢在未绑定玩家之前到达并干扰他们。飞行限制确保每个人都能公平地探索末地、寻找属于自己的鞘翅。
:::

## 绑定意味着什么

绑定后，这个鞘翅彻底属于你。来看看你能做什么、不能做什么：

### 你可以

| 操作 | 说明 |
|------|------|
| 正常飞行 | 装备在胸甲槽，使用烟花火箭加速 |
| 附魔 | 附魔台、铁砧、附魔书都可以正常使用 |
| 修复耐久 | 铁砧 + 幻翼膜修复，和原版一样 |
| 升级为下界合金鞘翅 | 锻造台升级，绑定信息自动继承 |
| 祛魔 | 研磨台可以移除附魔 |
| 放入末影箱 | 你自己的私有空间 |
| 放入随身背包 | 如果你有 Minepacks 随身背包 |

### 你不能

| 操作 | 说明 |
|------|------|
| 丢弃（按 Q） | 绑定的鞘翅不能丢在地上 |
| 放入箱子/潜影盒 | 不能放进公共容器 |
| 转交给其他玩家 | 别人捡不了你的鞘翅 |
| 挂到展示框/盔甲架 | 防止脱离你的控制 |
| 放入熔炉/漏斗等 | 一切公共容器都不行 |

简单记住：**绑定的鞘翅只能待在你身上、末影箱里、或随身背包里。**

## 死亡保护

你可能担心——万一死了，鞘翅会不会掉？

**不会。** 绑定鞘翅是灵魂绑定的，死亡后会自动保留。

- 正常死亡 → 重生后，鞘翅自动回到你的胸甲槽
- 如果胸甲槽有东西，会自动放进背包
- 如果背包也满了，鞘翅会掉在你脚下——**只有你能捡**

::: tip 提示
服务器开启了死亡不掉落（keepInventory），所以大部分情况下鞘翅根本不会离身。绑定保护是额外的一层安心保障。
:::

## 升级为下界合金鞘翅

有了下界合金升级模板和合金锭后，在锻造台升级鞘翅——绑定信息会自动转移到新的下界合金鞘翅上，不需要任何额外操作。

升级后会收到一条确认消息，新鞘翅上同样有你的名字和绑定日期。

## 查看绑定信息

在游戏中输入：

```
/elytrabind
```

可以查看你的绑定状态、绑定日期、以及鞘翅当前所在的位置。

## 如果鞘翅丢了

虽然绑定机制已经做了多重保护，但如果真的遇到意外（比如被某种极端bug吞掉），可以：

1. 先输入 `/elytrabind` 确认鞘翅状态
2. 如果显示「绑定的鞘翅丢失」，联系管理员
3. 管理员清除你的绑定记录后，你可以重新去末地城获取新鞘翅

## 小贴士

- **第一个鞘翅很重要** — 绑定是终身的，想好再拾取
- **先去末地城再附魔** — 先拿到绑定鞘翅，后续附魔修耐久都方便
- **展示框留给新人** — 如果你已经绑定过了，看到末地船的展示框请不要破坏，留给其他玩家
- **末地外岛禁飞** — 外岛无法使用鞘翅飞行，老老实实走路探索，公平竞争
- **配合飞行插件更安全** — 服务器还有飞行辅助机制，不用担心意外摔落
