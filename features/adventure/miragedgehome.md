# 星辉锚点

在锐界幻境的世界里，每一片土地都蕴含着星辰之力。你可以在冒险途中点亮沿途的星辉锚点，将它们刻入你的个人星轨。无论身在何处，只需一次传送即可瞬间回到标记的位置。

这不仅仅是一个「回家」的功能——星辉锚点系统让你的冒险足迹变得更有价值，每一次探索都在为你的星轨增添新的节点。

## 点亮你的足迹

### 私人星辉锚点

走到任何一个你想记住的位置，用 `/sethome` 就能点亮一个只属于你的星辉锚点。每个玩家默认可以设置 **1 个**锚点——请珍惜你的第一个锚点位，选一个真正重要的地方。

如果想要更多锚点，可以在社区活动中获取，或者找管理员申请增加锚点位上上限～

每个锚点都可以在背包界面中：
- 自定义名称和图标——把物品拖入图标槽，锚点就变成你喜欢的样式
- 添加描述文字——记录下这里有什么资源
- 随时传送回去——星轨校准几秒后，就能嗖地一声到达目的地

::: tip 冒险小贴士
在主基地或者常用的集结点设置星辉锚点，探索起来事半功倍。
:::

### 公开私人星辉锚点

如果你的锚点建在一个风景绝佳的观景点、一个资源丰富的矿洞入口、或者一个方便大家集合的位置，你可以把它开放为**公共星辉锚点**。

开放时可以设置每次访问的费用（灵叶），其他玩家传送过来时，费用会自动转入你的账户。这相当于在锐界幻境的版图上开设了一个小小的驿站。

> 注意：默认情况下你没有公开锚点的权限，如果需要使用公开功能，请联系管理员为你开放资格。

如果想取消公开，随时可以用 `/unpublish` 恢复为私人锚点。

## 公共星辉锚点网络

管理员会在全服设置一些官方公共锚点——比如各大主城、交通枢纽、活动场地等。这些锚点对所有玩家免费开放，方便大家快速到达服务器的各个角落。

你可以随时用 `/warps` 打开所有公共锚点列表，浏览服务器里有哪些好地方值得一去。

## 星辉信使——随时和朋友汇合

在冒险中，你可能会遇到这些情况：
- 朋友在远处发现了好东西，叫你快去看看
- 你在野外迷路了，想回到朋友身边
- 大家约好在某个位置集合，但有人走散了

这时候可以用**星辉信使**系统来解决：

- 用 `/tpa` 派出一位星辉信使，飞到朋友身边请求传送
- 用 `/tpahere` 邀请朋友传送到你这边
- 用 `/tpaui` 打开传送管理界面，一目了然查看所有待处理请求

> 每次成功传送会消耗 **100 灵叶**（如果安装了经济系统的话）。

### 被频繁打扰了？
如果不想被别人反复传送打扰，连续拒绝同一个人 3 次后，会自动开启一道星能屏障，5 分钟内对方无法再向你发送请求。既不会太严厉，也足够清静一下～

## 传送机制详解

星辉传送不是瞬间完成的，整个过程蕴含着一套完整的保护机制：

### 星轨校准

发送传送请求后，需要等待数秒进行星轨校准。这段时间内，**移动或受伤会打断校准**，传送取消。这是为了防止你在战斗中不小心点了传送时措手不及——校准被打断反而是保护你。

### 星辉扫描

传送前系统会对落点进行简单检查。如果发现目标位置不安全（虚空、岩浆等），你可以选择在 10 秒内再次执行命令，进行**强行跃迁**——不过后果自负哦～

### 传送后效果

成功传送后，你会短暂受到**缓慢和失明**效果的影响，持续几秒钟。这既是为了防止传送后立即被怪物围攻，也是一种谜一般的定位保护——让你稍微缓一缓，看清周围情况再行动。

### 星能冷却

每次成功传送后，需要休息几秒等待星能恢复。冷却期间再次传送会提示还需要等多久。

## 资源消耗

创建和公开星辉锚点需要消耗灵叶，这是为了让玩家珍惜自己的锚点位，而不是随意设置、用完就忘。

| 操作 | 费用（递增） | 说明 |
|------|------------|------|
| 创建第一个锚点 | 350 灵叶 | 第1个最便宜 |
| 如果还能设第2个 | +300 灵叶 | 每多一个递增 |
| 公开锚点（如有权限） | 900+ 灵叶 | 一次性费用 |
| 删除锚点 | **返还 60%** | 不会全亏 |
| 使用 /tpa 传送 | 100 灵叶/次 | 成功才扣费 |

这些费用的递增机制引导你只保留真正有用的锚点，而不是滥用资源。

::: tip 关于灵叶
灵叶是锐界幻境的基础货币，通过日常冒险、击杀怪物、参与活动获得。
:::

## 基岩版玩家看这里

完全不用担心操作问题。星辉锚点系统为基岩版玩家提供了专属的**表单界面**：

- `/homes` 弹出按钮列表，点击即可传送或设置
- `/warps` 分页展示所有公共锚点，翻页方便
- 收到 TPA 请求时弹出确认框，接受/拒绝一目了然
- `/tpaui` 打开传送管理表单
- 发送 TPA 时可以下拉选择在线玩家，不用记名字

Java 版和基岩版的功能完全对等，一个都不会落下。

## 和冒险的联动

星辉锚点系统不是孤立的。随着你在锐界幻境里的冒险深入，你会发现它和多个系统有自然的联动：

- **死亡轮回**：死亡后可以用 `/back` 返回死亡点，但走回去能捡回掉落的灵叶。如果你提前在附近设置过星辉锚点，走回去就方便多了
- **传送后效果**：传送后的缓慢+失明效果和死亡轮回中 `/back` 的负面效果机制一致——这是锐界幻境传送体系的一贯设计
- **公共锚点网络**：服务器会逐步在各个主要区域放置官方锚点，形成一张覆盖全服的传送网络
