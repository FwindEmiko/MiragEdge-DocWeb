# 附魔系统概览

> **锐界幻境的附魔体系**——原版经典 + 自定义拓展，200+ 附魔，7 个品质层级，全新获取模型。

## 获取途径

与原版最大的区别：**自定义附魔不从附魔台和村民处获得，全部通过探索获取。**

| 来源 | 可获取 |
|------|--------|
| 🔮 附魔台 | **仅原版附魔**（锋利/保护/效率等） |
| 🏘️ 村民交易 | **仅原版附魔** |
| 🗺️ 战利品箱子 | **全部自定义附魔**（遗迹/要塞/末地城等） |
| 🎣 钓鱼 | **低品质为主**（普通+优良+稀有≈98%） |
| 🎪 活动/BOSS | 史诗+传说+至宝品质 |

> 原版附魔仍可通过原版附魔台和村民正常获取，与自定义体系互不干扰。

## 核心机制

| 功能 | 说明 | 详情 |
|------|------|------|
| 附魔台 | 纯原版机制，不产出自定义附魔 | — |
| 铁砧 | 附魔合并、修复、改名 | [铁砧参数](#铁砧机制) |
| 砂轮 | 祛魔返还经验 | [砂轮系统](#砂轮系统) |
| 图鉴 | GUI 浏览全部附魔，支持多条件筛选 | 主城 NPC |
| 冲突 | 同类附魔互斥，详见分类页 | [附魔分类](./group.md) |

## 铁砧机制

关闭了原版的累积惩罚机制，采用简化的线性花费：

| 操作 | 花费 |
|------|:---:|
| 添加附魔（每级） | 2 经验/级 |
| 新增一种附魔 | +1 经验 |
| 改名 | 1 经验 |
| 修复物品 | 3 经验 |
| 最高花费上限 | 40 级 |

> 每件装备最多 16 个附魔，不同材质工具不可混拼。

## 砂轮系统

祛魔返还经验 = `30 × 等级/最大等级 × 品质倍率`

| 品质 | 倍率 |
|------|:---:|
| 普通 | 1.0x |
| 优良 | 1.2x |
| 稀有 | 1.5x |
| 史诗 | 2.5x |
| 传说 | 4.0x |
| 至宝 | 6.0x |
| 诅咒 | 不可祛魔 |

## 新手上路

1. **前期**（1-30级）：刷附魔台，获取原版附魔（锋利/保护/效率/耐久）
2. **中期**（30级+）：探索地牢、废弃矿井、沙漠神殿，获取自定义附魔
3. **后期**（满级装备）：挑战高级结构（要塞/末地城/堡垒），追求传说和至宝

## 注意事项

- ⚠ 附魔台**不会**产出自定义附魔——需要野外探索
- ⚠ 原版保护类附魔（保护/火焰保护/爆炸保护/弹射物保护）互斥
- ⚠ 锋利/亡灵杀手/节肢杀手互斥——根据目标选择
- ⚠ 精准采集和时运互斥
- 🔴 自定义附魔不会在村民交易中出现
