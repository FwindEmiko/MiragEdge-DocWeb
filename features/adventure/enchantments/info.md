# 附魔速览

> **本服务器的附魔分两块：原版附魔（和原版一样获取）+ 自定义附魔（需探索获得）。**  
> 下面的时间线告诉你不同阶段该关注什么。

## 新手阶段「跑图积累」

附魔台和村民交易**只出原版附魔**。自定义附魔需要去野外探索。

```
专注：锋利 / 保护 / 效率 / 耐久
来源：附魔台、村民、钓鱼 → 都属于原版
```

> 可以先用原版附魔玩着，不需要刻意刷自定义。

## 中期阶段「回收重铸」

当你有了一堆不想要的附魔书，就可以开启附魔循环了：

```
不要的附魔书
    ↓ 回收
魔法粉尘 ← 普通~诅咒品质
魔法结晶 ← 史诗~至宝品质
    ↓
重铸 → 未学习附魔书 → 学习（右键）→ 随机附魔书
融合 → 粉尘→结晶
增幅 → 提升单附魔书等级
```

> 详见 [附魔管理](./system.md)

## 后期阶段「追求稀世」

传说和至宝品质的附魔出现率极低（0.1%、0.06%），是长期追求目标。

```
史诗级  0.3%  ← 探索地牢/矿井就够
传说级  0.1%  ← 需要刷高级结构（要塞/末地城）
至宝级  0.06% ← 纯看脸 or 大量材料重铸
```
