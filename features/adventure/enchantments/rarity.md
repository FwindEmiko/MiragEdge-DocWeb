# 品质体系

本服附魔采用 **8 级品质** 分级机制，不同品质对应专属配色、权重和获取难度。

## 品质一览

| 品质 | 配色 | 色值 | 权重 | 概率 | 获取方式 |
|------|------|------|:---:|:---:|------|
| 普通 | 汉白玉 | `#f8f4ed` | 1000 | 56.2% | 附魔台、村民、钓鱼 |
| 优良 | 毛绿 | `#66c18c` | 500 | 28.1% | 附魔台、村民、钓鱼 |
| 稀有 | 霁青 | `#63bbd0` | 200 | 11.2% | 附魔台、村民、钓鱼 |
| 史诗 | 夹竹桃红 | `#eb507e` | 5 | 0.3% | 战利品箱、BOSS |
| 传说 | 淡橘橙 | `#fba414` | 2 | 0.1% | 战利品箱（稀有）、活动 |
| 至宝 | 油菜花黄 | `#fbda41` | 1 | 0.06% | 战利品箱（极稀有） |
| 诅咒 | 鹤顶红 | `#d42517` | 25 | 1.4% | 战利品箱 |
| 幻化 | 粉团花红 | `#ec9bad` | 1 | — | 特殊途径（仅粒子特效） |

## 权重机制

品质权重决定了附魔从战利品/钓鱼中出现的概率：

```
出现概率 = 该品质权重 / 所有品质权重之和

总权重 = 1000 + 500 + 200 + 5 + 2 + 1 + 25 = 1733

普通: 1000/1733 ≈ 57.7%
优良:  500/1733 ≈ 28.9%
稀有:  200/1733 ≈ 11.5%
史诗:    5/1733 ≈  0.3%
传说:    2/1733 ≈  0.1%
至宝:    1/1733 ≈  0.06%
诅咒:   25/1733 ≈  1.4%
```

::: tip 设计理念
品质分布呈**金字塔结构**——低品质为主力（普通+优良+稀有合计≈98%），高品质为稀有惊喜。钓鱼和战利品箱共享同一品质池，但高品质极难在钓鱼中出现。
:::

## 各品质详解

### 普通（权重 1000）
最基础的附魔品质，原版大部分附魔都属于此等级。

**代表附魔**：亡灵杀手、节肢杀手、爆炸保护、弹射物保护

### 优良（权重 500）
比普通稍好的进阶附魔。

**代表附魔**：火矢、火焰附加、荆棘、效率、耐久、水下速掘、击退

### 稀有（权重 200）
出现实用特效的附魔品质。

**代表附魔**：保护、力量、冲击、精准采集、水下呼吸、快速装填

### 史诗（权重 5）
强力附魔，需要探索才能获得。

**代表附魔**：时运、无限、处决、破军、暴戾、吸星、磁铁、夜伏、自生

### 传说（权重 2）
顶级附魔，可遇不可求。

**代表附魔**：经验修补、抢夺、捍卫、重铸、旺盛、自愈、脉络(禁用)、反重力

### 至宝（权重 1）
世间至宝，极度稀有。

**代表附魔**：索敌、立方、定力、咒怨、回溯

### 诅咒（权重 25）
带有负面效果的附魔，9 种诅咒效果各不相同。

### 幻化（权重 1，不可获取）
特殊品质，仅供粒子特效展示。无法从任何正常渠道获取。

## 获取策略

| 目标品质 | 推荐途径 |
|----------|----------|
| 普通/优良/稀有 | 附魔台 + 钓鱼（大量产出） |
| 史诗 | 探索地牢、废弃矿井、沙漠神殿 |
| 传说/至宝 | 要塞、末地城、堡垒遗迹、下界要塞 |
| 诅咒 | 任意战利品箱（1.4% 概率出现） |
| 幻化 | 活动/特殊奖励（无法自然获取） |

## 与旧版区别

本服附魔体系经过全面重设计，与原版/旧服的主要区别：

1. **品质名称更新**：罕见→优良、精良→稀有、传奇→传说、稀世→至宝、皮肤→幻化
2. **获取模型改革**：自定义附魔不再从附魔台和村民处获得，全部通过野外探索
3. **品质分布调整**：高品质权重大幅降低，传说/至宝更难获得
4. **附魔名称润色**：部分附魔名称优化，更直观易懂
5. **禁用冗余附魔**：慧眼、电锯、脉络（由其他插件/功能覆盖）
