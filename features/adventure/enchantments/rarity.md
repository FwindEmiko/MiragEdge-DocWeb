# 品质等级

附魔分成 **7 个品质等级**。品质越高越强，也越难获得。

## 一看就懂

| 品质 | 有多常见 | 回收能得到 |
|:----|:--------|:----------|
| 普通 | ⭐ 最常见 | 魔法粉尘 |
| 优良 | ⭐⭐ 常见 | 魔法粉尘 |
| 稀有 | ⭐⭐⭐ 还行 | 魔法粉尘 |
| 史诗 | ⭐⭐⭐⭐ 要碰运气 | 魔法结晶 |
| 传说 | ⭐⭐⭐⭐⭐ 很难得 | 魔法结晶 |
| 至宝 | ⭐⭐⭐⭐⭐⭐ 极稀有 | 魔法结晶 |
| 诅咒 | ⭐⭐⭐⭐ 看脸（负面效果） | 魔法粉尘 |

> 魔法粉尘和魔法结晶可以在[附魔管理](./system.md)系统里用来换新的附魔。

## 哪些附魔在什么品质

| 品质 | 典型附魔 |
|:----|---------|
| 普通 | 亡灵杀手、节肢杀手、爆炸保护 |
| 优良 | 火焰附加、荆棘、效率、耐久 |
| 稀有 | 保护、力量、精准采集、水下呼吸 |
| 史诗 | 时运、无限、吸星、磁铁、夜伏 |
| 传说 | 经验修补、抢夺、自愈、反重力 |
| 至宝 | 索敌（追踪箭）、立方（范围挖）、咒怨 |
| 诅咒 | 腐烂诅咒、脆弱诅咒 — 共9种 |
