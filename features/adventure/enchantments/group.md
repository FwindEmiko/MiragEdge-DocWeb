# 附魔分类与搭配

附魔按功能维度划分，不同类别之间存在**冲突互斥**关系。选对搭配，事半功倍。

## 冲突体系

本服附魔冲突分为 **6 大组**：

| 冲突组 | 成员 | 规则 |
|--------|------|------|
| 🔴 原版增伤 | 锋利 / 亡灵杀手 / 节肢杀手 / 致密 / 破甲 / 穿刺 | **互斥，只能选一个** |
| 🔴 原版防御 | 保护 / 火焰保护 / 爆炸保护 / 弹射物保护 / 捍卫 | **互斥，只能选一个** |
| 🔴 多重箭矢 | 火力 / 五言 / 三言 / 迭累 / 多重射击 | **互斥** |
| 🔴 范围挖掘 | 立方 / 贯穿 / 脉络(禁用) | **互斥** |
| 🔴 击退类 | 击退 / 击飞 / 迫近 / 威慑 | **互斥** |
| 🟢 生物增伤 | 屠宰 / 剿蛛 / 分解 / 破魔 / 圣水 / 终末 / 拆弹 | **可共存！**（但与锋利等冲突） |

::: tip 生物增伤可以叠加
剿蛛（蜘蛛伤害）+ 分解（史莱姆伤害）+ 圣水（亡灵伤害）可以共存于同一武器上。但它们与锋利/亡灵杀手等原版增伤互斥。
:::

## 实用搭配推荐

### 战斗剑士
```
🗡 剑：锋利V + 火焰附加II + 抢夺III + 耐久III + 处决III
🛡 胸甲：保护IV + 耐久III
👢 靴子：深海探索者III + 摔落缓冲IV + 耐久III
```

### 弓箭手
```
🏹 弓：力量V + 冲击II + 火矢I
（或）力量V + 索敌III（追踪箭）
```

### 矿工
```
⛏ 镐：效率V + 时运III + 耐久III + 吸星I
🦵 护腿：磁铁IV（自动吸物品）
```

### 建筑工
```
⛏ 镐：精准采集I + 效率V + 耐久III
👢 靴子：缓震III（防摔伤）
```
