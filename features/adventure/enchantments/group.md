# 分类与搭配

实际配装时该选什么附魔？附魔之间有冲突不能一起上，下面给你配好了直接抄。

## 冲突速查——哪些附魔不能共存

**1. 原版增伤**（互斥，只能选一个）
锋利 / 亡灵杀手 / 节肢杀手 / 致密 / 破甲 / 穿刺

**2. 原版防御**（互斥，只能选一个）
保护 / 火焰保护 / 爆炸保护 / 弹射物保护 / 捍卫

> 打不同怪换武器 vs 勤修装备更耐打——看你自己。捍卫是史诗级减伤附魔，比原版保护强但更难弄到。

**3. 多重箭矢**（互斥，只能选一个）
火力 / 五言 / 三言 / 迭累 / 多重射击

> 选一个你喜欢的手感。火力是散射，三言/五言是固定箭数。

**4. 击退类**（互斥，只能选一个）
击退 / 击飞 / 迫近 / 威慑

> 要不要把怪推开 / 拉近 / 打飞——看你的战斗风格。

**5. 范围挖掘**（互斥，只能选一个）
立方 / 贯穿

> 只能选一个范围挖矿方式。立方是四面八方挖，贯穿是直线挖。

**6. 生物特攻**（**可以共存！**）
屠宰（和平生物）/ 剿蛛（蜘蛛）/ 分解（史莱姆）/ 破魔（BOSS）/ 圣水（亡灵）/ 终末（末地怪）/ 拆弹（苦力怕）

> 一个剑上可以同时有蜘蛛伤 + 亡灵伤 + BOSS伤，打架时全部加算。  
> **但它们会和锋利/亡灵杀手冲突**——你只能用特攻附魔堆伤害，不能再用锋利了。

## 配装方案

### 日常战斗
```
🗡 剑：锋利V + 火焰附加II + 抢夺III + 耐久III + 处决III
🛡 胸甲：保护IV + 荆棘III + 耐久III
👢 靴子：深海探索者III + 摔落缓冲IV + 耐久III
```

### 远程弓手
```
🏹 弓：力量V + 冲击II + 火矢I（基础款）
🏹 弓：力量V + 索敌III（追踪箭—传说品质，很难拿）
```

### 专业矿工
```
⛏ 镐：效率V + 时运III + 耐久III + 吸星I
🦵 护腿：磁铁IV + 耐久III
```

> 吸星和磁铁搭配使用，挖矿时所有掉落物自动进包，不用弯腰捡。

### 建筑工
```
⛏ 镐：效率V + 精准采集I + 耐久III
👢 靴子：缓震III + 摔落缓冲IV
```

### 生存大师（后期毕业）
```
🗡 剑：利刃 + 夜伏 + 暴戾 + 会心 + 耐久 + 吸星 + 处决
⛏ 镐：效率V + 时运III + 耐久 + 吸星 + 洗炼
🛡 全套：保护IV + 耐久 + 重铸 + 自愈
👢 靴子：深海探索者III + 摔落缓冲IV + 飘逸
```
