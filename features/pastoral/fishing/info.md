# 更多钓鱼

> 锐界幻境为你带来了一套全新的钓鱼体验——稀有度分级、鱼竿进阶、特殊鱼饵、钓鱼比赛，甚至可以在岩浆和虚空里垂钓！

## 快速开始

1. **搞一根鱼竿** —— 进服找管理员领取「初学者的鱼竿」，或者自己用原版竿 + 铁锭合成
2. **找个水坑** —— 随便找个有水的地方，把鱼钩丢进去
3. **等鱼上钩** —— 浮漂下沉就右键收竿
4. **卖鱼赚钱** —— 打开指令菜单 `/emf`，把钓到的鱼卖掉换零花钱

> 一根初学者的鱼竿只能钓垃圾和普通品质的鱼。想要钓到更好的鱼，你需要升级你的装备。

## 核心机制

| 概念 | 说明 |
|:----|:-----|
| **稀有度** | 鱼分五个等级：垃圾、普通、稀有、史诗、传说。等级越高越罕见，价格也越高 |
| **鱼竿** | 决定你能钓到哪些稀有度的鱼。竿越好，能钓到的鱼越高级 |
| **鱼饵** | 附在鱼竿上可以提升特定鱼类的出现概率，让你能更有针对性地收集鱼类 |
| **比赛** | 每天多场钓鱼比赛实时开战，争夺排名拿奖励 |
| **维度钓鱼** | 中后期开放，岩浆和虚空都有专属鱼类 |
