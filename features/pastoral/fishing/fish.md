# 鱼类图鉴

> 服务器有超过 70 种自定义鱼类，涵盖五大稀有度。钓到的鱼可以卖钱，还能在图鉴中查看收集进度。

## 稀有度分级

| 稀有度 | 出现概率 | 价值范围 | 说明 |
|:-----|:--------|:---------|:-----|
| **垃圾** | 偶尔 | 不值钱 | 生锈的桶、破鞋、烂鱼竿……钓鱼佬的日常 |
| **普通** | 极其常见 | 几十块零花钱 | 鳕鱼、金枪鱼、鲤鱼等日常鱼类 |
| **稀有** | 较常见 | 几十到几百 | 鱿鱼、水母、各种蛙、电鳗等 |
| **史诗** | 罕见 | 几百到上千 | 海豚、企鹅、螃蟹、鲸鱼、美西螈 |
| **传说** | 极其罕见 | 上千到数万 | 国王鲭鱼、黄金蛙、蓝鲨、黑洞鱼……一发入魂 |

## 怎么卖鱼

1. 输入 `/emf` 打开主菜单
2. 点击「出售鱼类」
3. 把鱼放进左边的格子里
4. 点击出售按钮换钱

价格公式：**鱼的长度 × 稀有度倍率**，越长的鱼越值钱！

## 钓鱼图鉴

打开 `/emf` → 点击「钓鱼图鉴」，可以查看：
- 你钓过哪些鱼
- 每种鱼的最大尺寸、最小尺寸
- 首次发现的日期
- 全服最大/最小记录

未钓到的稀有度会显示为灰色锁图标，等你解锁！
