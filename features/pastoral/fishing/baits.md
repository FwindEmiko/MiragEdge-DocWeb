# 鱼饵系统

> 鱼饵可以附在鱼竿上，提高特定鱼类的出现概率。想在比赛中精准锁定目标鱼？鱼饵是你的好帮手。

## 怎么用

1. 对着空鱼竿按 Shift + 右键 → 打开鱼饵菜单
2. 把鱼饵拖到鱼竿上即可附饵
3. 下次钓鱼时鱼饵会自动生效，提升对应鱼类的出现率
4. 鱼饵有使用次数，用完就没了

也可以在 `/emf` 主菜单 → 鱼饵商店购买鱼饵。

## 鱼饵种类

| 鱼饵 | 效果 | 附在竿上的次数 |
|:----|:-----|:-------------|
| **虾仁** | 提升普通稀有鱼（鲤鱼、金鱼等）概率 | 最多 100 次 |
| **线虫** | 提升普通 + 传说鱼概率，万能型 | 最多 16 次 |
| **鲜活水蚤** | 提升史诗 + 传说鱼概率 | 最多 64 次 |
| **史诗灵药** | 专精提升史诗鱼概率 | 最多 64 次 |
| **传说诱饵** | 赌狗最爱——只提升传说鱼概率 | 最多 20 次 |
| **无尽鱼饵** | 提升传说鱼概率，**无限使用** | 用不完 |

## 怎么获得鱼饵

- **虾仁、线虫、水蚤**：在鱼饵商店直接购买
- **史诗灵药、传说诱饵**：价格较高，商店有售
- **无尽鱼饵**：不出售，只能通过特定活动或极低概率钓到

## 注意事项

- 每根鱼竿最多附 **7 种** 鱼饵
- 比赛期间鱼饵**不生效**，公平竞争
- 附了鱼饵的鱼竿在铁砧上会受到保护，防止被误吞
