## 快速上手

在 QQ 群 **523914625** 内，使用以下几组指令即可开始体验：

| 你要做什么 | 发送指令 |
| --- | --- |
| 绑定游戏账号 | 进服务器获取验证码 → 群内发 `绑定xxxx` |
| 查看帮助菜单 | `菜单` |
| 和星玖聊天 | `@星玖` + 你想说的话 |
| 查自己信息 | `我的信息` |
| 发送群服消息 | `#` + 你要说的话 |

## 绑定管理

### 绑定流程

入服需要先绑定社交账号，两步完成：

1. **进入服务器**，会弹出绑定提示，上面有一个 **4 位数字验证码**
2. 在群内发送 `绑定<验证码>`（例如 `绑定1234`）即可完成绑定

绑定成功后就可以自由进出服务器了。

### 解绑

在原绑定群内发送：

```
解绑 ID
```

>  退出交流群也会自动解绑

## 背包预览

在群内直接查看**自己**的背包物品，以渲染图片形式呈现，所见即所得。

**指令：**

```
我的背包
```

## 末影箱预览

查看**自己**末影箱中的全部物品，以渲染图片展示。无论在线与否均可查询。

**指令：**

```
我的末影箱
```

> 支持大末影箱（54格）。

## 潜影盒预览

查看**自己**背包中潜影盒内的物品。

**指令：**

```
我的盒子
```

若背包中有多个潜影盒，会自动列出供你选择查看哪一个：

- **潜影盒 ≤ 10 个：** 支持点击表情或发送数字序号选择
- **潜影盒 > 10 个：** 仅支持发送数字序号选择

支持铁砧重命名的自定义名称显示。支持大潜影盒（54格）。

## 解封申请

在游戏中因违规被封禁的玩家，可通过群内申请解封。

**指令：**

```
解封 <游戏ID>
```

>  解封申请将由管理员审核，请说明原因。恶意刷屏会被无视。

## 注意事项

- 绑定相关操作请勿将验证码告知他人
- 排行榜数据尽可能实时更新
- 背包/末影箱/潜影盒预览渲染图片可能需要几秒生成时间
